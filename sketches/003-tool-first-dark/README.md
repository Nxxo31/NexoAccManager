# NX-Manager - Tool First Dark Mockup

## Stance Description
This mockup represents a **tool-first, dark-mode focused** design approach that emphasizes:
- **Tool-centric interface**: Primary actions are prominent and easily accessible
- **Dark theme first**: Designed primarily for dark mode with light mode as secondary
- **Tool-like aesthetic**: Inspired by professional developer tools and creative applications
- **High information density**: Maximum utility per pixel
- **Professional feel**: Clean, precise, and purposeful

## Design Decisions

### Toolbar
- Minimal header with only essential controls: theme toggle (🌙☀️) and settings (⚙️)
- Compact height to maximize vertical space
- Clear visual hierarchy with subtle hover effects

### Left Slicer (Accounts)
- **Search bar**: Prominent input with clear placeholder and search icon
- **Add Account button**: Primary action button with "+" icon in Roblox Red
- **Show/Hide toggle**: Clean switch for filtering visibility
- **Account list**:
  - Tool-inspired account rows with drag handles (cursor-grab)
  - Compact avatar placeholder with "RBX" badge
  - Account name and description in clean typography
  - Inline group editing via colored tags (simulated)
  - Visual drag indicators via cursor-grab styling
  - Status indicators with colored dots and relative timestamps
  - Maximum information density: shows multiple tags per account with clear visual separation

### Main Content Area
- Clean header with section title
- Welcome state with clear calls-to-action
- Quick stats dashboard showing key metrics
- Tool-panel aesthetic with subtle elevations and borders

## Technical Implementation Notes
- Uses Tailwind CSS via CDN for rapid prototyping
- Custom CSS variables for comprehensive theme management (dark/light)
- Font Awesome 6 for consistent, high-quality icons
- Basic JavaScript for theme persistence and drag simulation
- All interactive elements have deliberate hover and focus states
- Designed to feel like a professional tool rather than a consumer app

## Intended User Experience
Targeted at users who:
- Work with Roblox accounts professionally or semi-professionally
- Spend extended periods in the application
- Prefer dark interfaces for reduced eye strain
- Value efficiency and direct access to tools
- Are comfortable with slightly higher information density in exchange for power
- Appreciate tool-like interfaces (similar to IDEs, design tools, or professional software)

This stance balances power and usability, providing direct access to core functions while maintaining a clean, professional aesthetic that feels purpose-built for account management.
